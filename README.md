# GomenneBullet_Project
さみしがりやのエルフに愛を伝えるシューティングゲーム\
unityroom → https://unityroom.com/games/gomennnebullet    
さみしがりやのエルフは、お互いのメンタルの状態を可視化できるように領域を展開する。\
主人公のライフ(メンタル)が壊れる前に、エルフに愛を伝えきる。\
愛を伝えきれたら(ゲージが満タンになったら)ゲームクリア\
お怒りを受け、主人公のメンタルが壊れてしまったらゲームオーバー\
操作方法はwasd or 矢印キー　使いやすい方を選択できる。\
HPが０か、ゲージが満タンになったらクイズが発生。\
【体制】個人開発　【担当範囲】企画、キャラクターデザイン・イラスト制作、Live2Dモデリング、Unity（C#）を用いたプログラミング全般\
【使用ツール・技術】Unity, C#, Live2D Cubism，ClipStudio\
【開発補助ツール】Gemini（エラー箇所の特定や、未経験の実装アプローチの調査など）\
【使用アセット・外部素材】\
Live2DのUnity組み込み処理，モデルの視線追従処理：Live2D Cubism SDK for Unity を使用\
一部UI素材（矢印キー）：イラストAC（No: 26287943）\
【工夫点】\
・直感的なUIとシームレスなゲーム導入\
　プレイヤーがルールに迷わず、すぐにプレイへ没入できるよう開始前のカウントダウンを実装しました。\
　また、タイトル画面の「スタートボタン」をそのままゲーム本編の操作ボタンとして機能させることで、画面移行による思考の分断を防ぎ、爽快なプレイ体験を実現しています。\
・タイマー実装によるリプレイ性の向上\
　単なるクリア目的だけでなく、「いかに早くクリアするか」というタイムアタック要素を取り入れるため、タイマー機能を実装しました。\
　プレイヤーごとの多様な楽しみ方を提示し、繰り返し遊べる設計にしております。\
・世界観を深めるクイズ機能による難易度調整\
　単調なシューティングゲームに陥るのを防ぎ、作品のユニークな世界観を維持するため、ゲームオーバー直前（救済措置）およびクリア直前にクイズ機能を実装しました。\
　プレイヤーが理不尽にすぐ負けてモチベーションを下げることのないよう、システム面から難易度のバランスを調整しています。
